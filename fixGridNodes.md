# fixGridNodes

## Description:  
Fixes GridNodes that lie behind a surface  

## Syntax:  
>MPM.fixGridNodes(GridName='----', PointOnSurface='----', SurfaceNormal='----')  

### Datatypes:  
*GridName - string  
*PointOnSurface - 3 x 1 double  
*SurfaceNormal - 3 x 1 double  

>**Note**  
>Do not fix the extreme planes of the Grid

# addRigidBody

## Description:  
*Returns properties of rigid body  
*Sets property of a rigid body  

## Syntax:  
### Return:  
>MPM. RigidBodyProperties (BodyName ='***’)  

### Datatypes:  
*BodyName - string  

### Set:  
>MPM. RigidBodyProperties (BodyName ='***’, PropertyName='***’, Value='***’)  

### Datatypes:  
*BodyName - string  
*PropertyName - string
*Value - double

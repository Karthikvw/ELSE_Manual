# addSolver      

## Description:  
Add a Solver to the process     

## Syntax:  
>MPM.addSolver(SolverName='----', SolverKind='----', TStart='----', TEnd='----', TInc='----')  

*TStart - Start time*  
*TEnd - End time*  
*TInc - Time increment*  

### Datatypes:  
*SolverName - string  
*SolverKind - string  
*TStart - double  
*TEnd - double  
*TInc - double    

>**Note**
>SolverName and SolverKind are to be chosen only from the available list
